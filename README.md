# María Teresa Sánchez-Seco Alcocer - Web Profesional

Despacho de abogada en Alcalá de Henares.

## Estructura
- /src/components/site - Componentes reutilizables (Header, Footer, Forms, etc.)
- /src/routes - Rutas de la aplicación (index, aviso-legal, politica-privacidad, etc.)
- /src/assets - Imágenes y recursos multimedia
- /public - Archivos estáticos

## Páginas incluidas
- Landing page principal
- Aviso legal
- Política de privacidad  
- Política de cookies

## Funcionalidades
- Formulario de contacto
- Formulario de envío seguro de documentos
- Mapa de Google Maps integrado
- Botón flotante de WhatsApp
- Diseño responsive móvil/desktop

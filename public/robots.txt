# 🏛️ María Teresa Sánchez-Seco Alcocer - Web del Despacho

Sitio web profesional para el despacho de abogacía de María Teresa Sánchez-Seco Alcocer, ubicado en Alcalá de Henares.

## 🚀 Tecnologías

- **Framework**: React 18 + TypeScript
- **Router**: TanStack Router
- **Estilos**: Tailwind CSS v3
- **Iconos**: Lucide React
- **Build**: Vite
